# ZIP-Compressor
Implementation of DEFLATE (LZ77 + Huffman) from Scratch in C++.
(A lossless compression engine from scratch using LZ77 and Huffman coding)

## Overview
Implementation of LZ77-based compression algorithm from scratch in C++.

## Features
- LZ77 Compression
- LZ77 Decompression
- Modular architecture
- Sliding window implementation

## Folder Structure
(include project tree)

## How to Run
g++ src/lz77.cpp main.cpp -o compressor
./compressor
